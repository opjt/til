CNCF (Cloud Native Computing Foundation) ​

CNCF는 Cloud Natvie Computing Foundation 의 약자로,
클라우드 네이티브 기술 생태계를 표준화하고 발전시키는 재단이다. (비영리)

클라우드 네이티브 란? ​

클라우드 네이티브란 클라우드 컴퓨팅 환경에서 현대적 어플리케이션을 구축, 배포 및 관리할 때의 소프트웨어 접근 방식이다

쉽게 말해 클라우드를 사용하는 것에 그치지 않고 클라우드 환경에 맞춤 설계된 방식을 의미.(마치 클라우드환경에서 태어난것처럼)

CNCF 아웃풋 ​

잘 알려진 프로젝트로는 컨테이너 오케스트레이션 플랫폼인 Kubernetes가 있음.

https://www.cncf.io/projects/

위 링크를 보면 CNCF의 졸업 작품들을 볼 수 있음.

CNCF 프로젝트 단계 ​

Sandbox

  • 이제 막 실험적으로 CNCF에 들어온 프로젝트

Incubating

  • 기술적 잠재력이 인정받아 발전 중인 프로젝트(아기가 인큐베이터 안에 있는 형상 )

Graduated

  • 공식 표준급 프로젝트
