н•ҳмқҙнҚјл°”мқҙм Җ Hypervisor вҖӢ

window VMWareлЎң vmмқ„ мҳ¬лҰ¬лҠ” кІғкіј proxmoxм—җм„ң vmмқ„ лқ„мҡ°лҠ” кІҢ лӯҗк°Җ лӢӨлҘҙмЈ  ?

  • н•ҳмқҙнҚјл°”мқҙм ҖлһҖ? - RedHat

н•ҳмқҙнҚјл°”мқҙм ҖлҠ” м»ҙн“Ён„° лҰ¬мҶҢмҠӨлҘј к°ҖмғҒлЁёмӢ (VM)м—җ н• лӢ№н•ҳлҠ” мҶҢн”„нҠёмӣЁм–ҙмқҙлӢӨ.
мқҙ кё°мҲ лЎң лӢЁмқј лЁёмӢ (л¬јлҰ¬м Ғ)м—җм„ң м—¬лҹ¬ VMмқ„ мғқм„ұн•ҳм—¬ мӢӨн–үн•ҳлҠ” к°ҖмғҒнҷ”к°Җ к°ҖлҠҘн•ҳлӢӨ.

лӢӨлҘё л§җлЎңлҠ” к°ҖмғҒнҷ” лЁёмӢ  л§ӨлӢҲм Җ(лӘЁлӢҲн„°) (Virtual machine manager,monitor) лқјкі лҸ„ н•Ё.

л¬јлҰ¬м Ғ мһҗмӣҗ(CPU,RAM,л””мҠӨнҒ¬ л“ұ)мқ„ мӘјк°ңм„ң к°Ғ к°ҖмғҒлЁёлӢҲ(VM)м—җкІҢ лӮҳлҲ мЈјлҠ” к°җлҸ…кҙҖ м—ӯн• мқҙлқјкі  ліҙл©ҙ лҗңлӢӨ.

  • VM мһ…мһҘм—җм„  мһҗмӢ мқҙ м§„м§ң н•ҳл“ңмӣЁм–ҙлҘј м“°кі  мһҲлӢӨкі  м°©н•ҳкІҢ л§Ңл“¬

н•ҳмқҙнҚјл°”мқҙм Җ Type1, Type2 вҖӢ

н•ҳмқҙнҚјл°”мқҙм Җк°Җ н•ҳл“ңмӣЁм–ҙлҘј м§Ғм ‘ м ңм–ҙн•ҳлҠ”м§Җ,
м•„лӢҲл©ҙ нҳёмҠӨнҠёOS мң„м—җм„ң м ңм–ҙн•ҳлҠҗлғҗм—җ л”°лқј Typeмқҙ лӮҳлүҳкІҢ лҗңлӢӨ.

Type 1 (Bare-Metal)

н•ҳл“ңмӣЁм–ҙ л°”лЎң мң„м—җ н•ҳмқҙнҚјл°”мқҙм Җк°Җ мҳ¬лқјк°ҖлҠ” л°©мӢқмқҙлӢӨ.
лІ”мҡ© мҡҙмҳҒмІҙм ң(Windows, macOS л“ұ)м—Ҷмқҙ н•ҳмқҙнҚјл°”мқҙм Җк°Җ н•ҳл“ңмӣЁм–ҙлҘј м§Ғм ‘ м ңм–ҙн•ҳкі  к°ҖмғҒлЁёмӢ л“Өмқ„ кҙҖлҰ¬н•ңлӢӨ.

  • л№Ҳ н•ҳл“ңмӣЁм–ҙ(bare-metal)м—җ н•ҳмқҙнҚјл°”мқҙм Җк°Җ мІ«лІҲм§ёлЎң мҳ¬лқјк°ҖлҠ” нҳ•нғңмқҙкё° л•Ңл¬ём—җ лІ м–ҙл©”нғҲ нҳ•мӢқмқҙлқј н•Ё
  • нғҖмһ…1 н•ҳмқҙнҚјл°”мқҙм ҖлҸ„ кё°мҲ м ҒмңјлЎ  мқјмў…мқҳ мҡҙмҳҒмІҙм ңмқҙкұ°лӮҳ, мөңмҶҢн•ңмқҳ OS кё°лҠҘмқ„ нҸ¬н•Ён•ҳкі  мһҲлӢӨ.
    • мқјл°ҳм Ғмқё лІ”мҡ© мҡҙмҳҒмІҙм ңк°Җ м•„лӢҢ к°ҖмғҒнҷ”м—җ нҠ№нҷ”лҗң кІҪлҹүOSлқјкі  ліҙл©ҙ лҗЁ.

Type 2 (Hosted)

host OS(Windows, macOS л“ұ мқјл°ҳ мҡҙмҳҒмІҙм ң) мң„м—җ н•ҳмқҙнҚјл°”мқҙм Җк°Җ мҳ¬лқјк°ҖлҠ” л°©мӢқмқҙлӢӨ.
н•ҳл“ңмӣЁм–ҙ мң„м—җ host OSк°Җ мқҙлҜё м„Өм№ҳлҗҳм–ҙ мһҲкі , н•ҳмқҙнҚјл°”мқҙм Җк°Җ к·ё мң„м—җ мҳ¬лқјк°Җм„ң лҸҷмһ‘н•ҳкІҢ лҗңлӢӨ.

  • нҳёмҠӨнҠёOSлҘј кұ°міҗм„ң н•ҳл“ңмӣЁм–ҙм—җ м ‘к·јн•ҳкё° л•Ңл¬ём—җ м¶”к°Җм Ғмқё мҳӨлІ„н—Өл“ңк°Җ л°ңмғқн•ңлӢӨ.
  • лҢҖн‘ңм ҒмңјлЎң VMWare Workstation, Oracle VirtualBox л“ұмқҙ мһҲлӢӨ.

proxmoxлҠ”? вҖӢ

к·ёл ҮлӢӨл©ҙ proxmoxлҠ” нғҖмһ…1мқёк°Җ нғҖмһ…2мқёк°Җ?
кІ°лЎ л¶Җн„° л§җн•ҳмһҗл©ҙ нғҖмһ…1мқҙлӢӨ.

type1мқҖ OSк°Җ м—ҶлӢӨлҠ” лң»мқҙ м•„лӢҲлқј, н•ҳмқҙнҚјл°”мқҙм Җк°Җ н•ҳл“ңмӣЁм–ҙмқҳ мЈјмқёмқҙлғҗк°Җ кё°мӨҖмқҙкё° л•Ңл¬ё.

  • proxmoxлҠ” лҚ°л№„м•Ҳ лҰ¬лҲ…мҠӨлҘј кё°л°ҳмңјлЎң н•ҳм§Җл§Ң мқҙлҠ” мқјл°ҳм Ғмқё лҚ°л№„м•Ҳмқҙ м•„лӢҢ к°ҖмғҒнҷ” м „мҡ©мңјлЎң м»ӨмҠӨн…Җлҗң OSкі  мқҙкІҢ мҰү н•ҳмқҙнҚјл°”мқҙм Җлқјкі  н•  мҲҳ мһҲмқҢ.
