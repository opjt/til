# Unix Domain Socket

가끔 `.sock` 파일을 본적이 있는데 이게 `unix domain socket` 을 나타내는 파일 형태의 소켓 엔드포인트이다.

유닉스 도메인 소켓이란

- 동일한 호스트(로컬 시스템)내에서 실행되는 프로세스 간 통신(IPC) 를 위해 사용되는 메커니즘 중 하나로
- IPC 소켓 또는 로컬 소켓이라고도 함

네트워크 소켓(TCP/IP)이 IP주소와 포트번호를 사용하는 것과 달리, **UDS**는 로컬 파일 시스템 경로를 소켓 주소로 사용하여 식별됨.

TCP/IP 네트워크 스택 전체를 건너뛰기 때문에 로컬에서 더 나은 속도를 기대할 수 있음.

파일시스템 권한으로 접근을 제어할 수 있다는 이점도 있음
